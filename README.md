# 🌌 Aurora | Plataforma de Gestão de Projetos Técnicos

[Link do Projeto](https://www.figma.com/design/9wQSNoJRL2nHEvTIR4Reqk/Projeto-Interface-M%C3%A9dia-Fidelidade?node-id=0-1&p=f&t=X4ZJeZHnDtiDcVEW-0)

## 📖 Sobre o Projeto
O **Aurora** é uma plataforma interna de gestão projetada para empresas de tecnologia. O foco do sistema é organizar a alta densidade de dados de projetos multidisciplinares, facilitando o fluxo entre desenvolvedores.

Este projeto foi desenvolvido como parte de um desafio de interface desktop de média fidelidade, priorizando a **clareza organizacional**, **fluidez operacional** e **suporte à colaboração técnica**.

---

## 🎯 Objetivos Do Projeto
A interface permite que as equipes realizem a gestão completa do ciclo de vida de um software:
- **Gestão Técnica:** Criação, edição e organização de tarefas.
- **Atribuição Inteligente:** Definição de responsáveis e prazos.
- **Visibilidade de Progresso:** Acompanhamento visual de status e evolução dos projetos.
- **Controle de Dependências:** Mapeamento de tarefas para evitar atrasos.

---

## 🛠️ Funcionalidades Mapeadas

### Tarefa Principal
- **Fluxo de Criação:** Interface otimizada para criar tarefas, atribuir a membros e definir prioridades em um único fluxo.

### Tarefas Secundárias
- **Edição e Reatribuição:** Flexibilidade para mudar prioridades ou responsáveis conforme a demanda.
- **Filtros Avançados:** Busca rápida de tarefas por Status.
- **Detalhamento Completo:** Visualização de descrições técnicas, histórico de alterações e área de comentários para colaboração.
